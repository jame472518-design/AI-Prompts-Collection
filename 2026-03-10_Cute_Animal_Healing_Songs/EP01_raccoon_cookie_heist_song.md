**對應影片：** EP01 The Cookie Heist (餅乾大盜)
**風格：** 療癒系輕民謠 / 睡前故事感
**BPM：** 85-95
**配器建議：** 木吉他 + 鐘琴 + 輕拍手

## 演唱建議

| 項目 | 建議 |
|------|------|
| 速度 | 慢～中速，BPM 85-95 |
| 風格 | 輕民謠 / 療癒系，像唱給小朋友的睡前故事 |
| 配器 | 木吉他 + 鐘琴 + 輕拍手，簡單就好 |
| 語氣 | Verse 用說故事的感覺，Chorus 稍微放開一點 |
| Bridge | 語速稍快，帶一點驚慌的俏皮感 |
| Outro | 越來越小聲，像快睡著了 |

## MV 概念

可搭配 EP01 的可愛動物影片作為 MV，形成 Shorts（故事）→ 長影片（歌曲 MV）的內容循環。
