# CD01-EP01「雨夜」

**系列：** 回家的路
**時長：** 約 35 秒
**情緒曲線：** 孤獨 → 一點點希望

---

## 場景 1（10 秒）— 避雨

### A. Qwen Custom Prompt

```
請以輸入的圖片作為起始幀，生成一個10秒的3D Pixar風格動畫片段。

主角是一隻 chibi 比例的小橘貓，頭大身體小，超大琥珀色圓眼帶光澤反射，粉紅三角鼻，蓬鬆橘色毛帶淡虎斑紋，左耳有小缺口，尾巴末端較深。目前是流浪狀態，毛髮稍凌亂，微微沾溼。

鏡頭構圖：中景鏡頭，採用電影級淺景深構圖，主角位於畫面右三分之一處。紙箱佔據下方畫面，巷弄延伸至背景形成縱深。
視頻氛圍：冷調藍紫色光線，雨夜城市，潮溼的柏油路面反射零星路燈光點，氣氛寂寞壓抑但隱含一絲未知的溫暖。整體色調偏暗，僅紙箱邊緣有微弱的暖色反光。
動作描述：小橘貓蜷縮在破舊紙箱下避雨，把自己縮得很小很小。耳朵緊貼頭頂，大眼睛不安地微微左右張望。身體每隔幾秒因寒冷而輕微顫抖一下，然後慢慢把頭埋進自己的前爪裡，像是想把自己藏起來。
運鏡：鏡頭從小橘貓臉部特寫開始（能看到溼漉漉的毛髮和顫抖的鬍鬚），緩慢 dolly out 拉遠到中景，展現紙箱和雨夜巷弄全貌。鏡頭拉遠後，小橘貓在畫面中顯得格外渺小孤單。

## 場景 2（10 秒）— 被忽略

### A. Qwen Custom Prompt

```

鏡頭構圖：低角度鏡頭（貓的視角），畫面上半部是匆匆走過的行人雙腳和雨傘，下半部是小橘貓仰望的臉。構圖強調體型對比——行人的腳和巨大雨傘 vs 渺小的貓。
視頻氛圍：持續的冷色雨夜，但行人身上帶有略暖的路燈反光，形成對比。氛圍從寂寞轉為一絲期待再回到失落。
動作描述：多組行人的腳步和雨傘快速經過畫面（由左至右、由右至左交替），小橘貓的超大眼睛像看網球一樣左右追蹤每雙腳。突然一雙腳步放慢，小橘的耳朵瞬間豎起、眼睛亮起期待的光芒、身體微微前傾——但那雙腳猶豫一秒後加速走遠。小橘的耳朵緩緩垂下，眼神黯淡，身體縮回原位。
運鏡：固定低角度鏡頭，不移動，讓行人的腳步在畫面中穿梭。當那雙放慢的腳步靠近時，鏡頭微微 push in 到小橘臉部，捕捉期待的表情；腳步走遠後，鏡頭停留在小橘失望的臉上。

## 場景 3（10 秒）— 那道光

### A. Qwen Custom Prompt

```
請以輸入的圖片作為起始幀，生成一個10秒的3D Pixar風格動畫片段。

主角是一隻 chibi 比例的小橘貓，頭大身體小，超大琥珀色圓眼帶光澤反射，粉紅三角鼻，蓬鬆橘色毛帶淡虎斑紋，左耳有小缺口，尾巴末端較深。目前全身溼透，蜷縮在紙箱旁。

鏡頭構圖：側面中景，小橘貓位於畫面左側前景，遠處巷弄盡頭右上方一扇窗戶亮著溫暖橘黃色光芒。構圖形成從冷色（左）到暖色（右）的漸變，暗示希望的方向。
視頻氛圍：雨勢明顯減小變成細雨，空氣中瀰漫霧氣。整體依然偏冷色調，但遠處窗戶的暖光成為畫面中最醒目的色彩焦點，像黑暗中的燈塔。氛圍從沉寂轉向隱約的希望。
動作描述：小橘貓慢慢抬起頭，雨絲落在牠的臉上，牠微微眯眼。接著牠的視線被遠處那扇窗的光吸引——瞳孔慢慢放大，耳朵從貼平一點一點豎起來。牠的頭微微歪向一側，像是在確認那道光是不是真的。溼漉漉的大眼睛裡清楚映出那道橘黃色的光芒，眼睛變得水汪汪的。
運鏡：從中景開始，當小橘注意到光時，鏡頭緩慢 dolly in 推進到小橘臉部特寫，最後定格在牠映著光芒的眼睛超近景（能看到瞳孔中的窗戶倒影）。
配音描述：雨聲明顯減弱變成淅瀝的細雨聲，遠處傳來幾聲清脆的風鈴聲（從那扇窗的方向）。配樂加入溫暖的弦樂，與鋼琴交織，旋律微微上揚帶來一絲希望。旁白以溫暖男聲、使用繁體中文台灣口音，聲音更輕更柔地說：「但是那道光……好溫暖。」

注意：不要生成任何字幕、文字覆蓋或螢幕上的文字。
```

## 結尾 Hook（5 秒）

### A. Qwen Custom Prompt

```
請以輸入的圖片作為起始幀，生成一個5秒的3D Pixar風格動畫片段。

鏡頭構圖：側面全身中景，小橘貓位於畫面中央偏左，面朝右方（光的方向）。地面是溼漉漉反光的路面，遠處暖光模糊可見。
視頻氛圍：雨幾乎停了，細雨如薄霧。地面水窪反射著遠處的暖光。氣氛從靜止轉為一個微小的行動時刻，帶著不確定但義無反顧的感覺。
動作描述：小橘貓從蜷縮的姿勢慢慢撐起身體站起來，身體微微搖晃（因為虛弱和溼冷）。牠深吸一口氣，目光堅定地看向前方，然後猶豫地朝那道光的方向邁出第一步——一隻小爪子踏進水窪，濺起微小的水花。就在爪子落地的瞬間，畫面定格。
運鏡：低角度側面鏡頭，跟隨小橘站起的動作微微上移，然後當牠邁步時鏡頭靜止不動，讓小橘走向畫面右側。在爪子踏入水窪濺起水花的瞬間，鏡頭快速 push in 到爪子和水花的特寫，定格。
配音描述：環境聲幾乎靜音，只有極輕的水滴聲。當小橘站起時，配樂的弦樂輕輕揚起一個未完成的上行旋律。爪子踩入水窪的「啪嗒」聲清晰響亮。旁白以溫暖男聲、使用繁體中文台灣口音，帶著期許說：「牠會找到回家的路嗎？」然後畫面漸暗，音樂漸弱。

注意：不要生成任何字幕、文字覆蓋或螢幕上的文字。
```
