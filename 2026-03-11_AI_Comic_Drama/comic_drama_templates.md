# AI Comic Drama Story Templates

> Healing Animal-Themed Comic Drama for YouTube Shorts / Reels
> Total Duration per Episode: ~70-80 seconds (8 scenes x 8-10s each)

---

## How to Use

### Overview

This file contains **3 ready-to-use story templates** for producing AI-generated comic dramas (AI 漫劇). Each template is a complete mini-story with 8 scenes designed for vertical short-form video (9:16 aspect ratio).

### Production Pipeline

1. **Image Generation** -- Use the **Image Prompt** for each scene in ComfyUI (or Stable Diffusion / Midjourney). Generate at 1080x1920 resolution.
2. **Add Motion** -- Apply the **Motion Effect** using tools such as PixVerse, Runway Gen-2, or After Effects (puppet warp / particle effects).
3. **Camera Movement** -- Apply **Ken Burns** pan/zoom in your video editor (CapCut, Premiere Pro, DaVinci Resolve).
4. **Voiceover** -- Record or TTS-generate the **Narration Script** (Chinese). Recommended TTS: Azure TTS (Xiaoxiao / Yunxi voice) or ElevenLabs.
5. **Audio** -- Layer in the **Audio Cue** (SFX + BGM) from royalty-free libraries (Pixabay, Artlist, Epidemic Sound).
6. **Assembly** -- Arrange all 8 scenes sequentially. Add subtitles matching the narration. Export at 1080x1920, 30fps.

### Prompt Framework (Base Style)

All image prompts in this file share a common style prefix. Prepend this to every prompt if your workflow requires a full standalone prompt:

```
chibi kawaii [CHARACTER], oversized head, tiny body, sparkly eyes, plush texture,
3D Pixar style, soft pastel lighting, cinematic composition, vertical 9:16 aspect ratio,
high detail, 8k render, Unreal Engine 5 quality
```

### Scene Structure

| Act       | Scenes | Purpose          |
|-----------|--------|------------------|
| Setup     | 1 - 2  | 鋪陳：建立角色與場景 |
| Event     | 3 - 4  | 事件：引發轉折的關鍵 |
| Climax    | 5 - 6  | 情感高潮：最觸動人心的瞬間 |
| Ending    | 7 - 8  | 結局 + 感悟金句   |

### Timing Guide

- Each scene: **8-10 seconds**
- Total: **~70-80 seconds**
- Narration pacing: ~3-4 Chinese characters per second for a warm, gentle tone
