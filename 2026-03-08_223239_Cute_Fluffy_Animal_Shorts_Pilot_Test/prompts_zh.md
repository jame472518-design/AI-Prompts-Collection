# Pilot Test - 可愛毛茸茸動物角色（卡通風格）

## 測試計畫
- 每種動物：1 張縮圖 + 3 個影片情境（每個 = 2 段 x 6秒 = 12秒）+ 3 個音樂提示詞
- 風格：誇張可愛卡通，不要寫實
- 模型：Wan 2.1/2.2

---

## 風格關鍵字（所有 prompt 都要加上）

### 必須包含（可愛卡通風格）：
`3D cartoon Pixar style, chibi proportions, oversized head, exaggerated big sparkly eyes, tiny body, extremely fluffy and soft fur, round chubby cheeks, kawaii aesthetic, pastel color palette, plush toy texture, soft cel shading, adorable character design`

### 負面提示詞（全域套用）：
`realistic, photorealistic, uncanny valley, dark, scary, horror, sharp teeth, thin fur, skinny, angular features, blurry, distorted, extra limbs, deformed`

---

## Wan 影片提示詞規則
1. 每段 6 秒片段 80-120 字（英文）
2. 結構：主體 → 動作 → 環境 → 鏡頭
3. 每段片段只用一個鏡頭運動
4. 片段 1 = 鋪墊，片段 2 = 爆點/笑點

---
