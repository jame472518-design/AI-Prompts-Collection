# Pilot Test - Cute Fluffy Animal Characters (Cartoon Style)

## Test Plan
- Each animal: 1 thumbnail image + 3 video scenarios (each = 2 clips x 6sec = 12sec) + 3 music prompts
- Style: exaggerated cute cartoon, NOT realistic
- Model: Wan 2.1/2.2

---

## Style Keywords (add to ALL prompts)

### Must Include (cute cartoon style):
`3D cartoon Pixar style, chibi proportions, oversized head, exaggerated big sparkly eyes, tiny body, extremely fluffy and soft fur, round chubby cheeks, kawaii aesthetic, pastel color palette, plush toy texture, soft cel shading, adorable character design`

### Negative Prompt (apply globally):
`realistic, photorealistic, uncanny valley, dark, scary, horror, sharp teeth, thin fur, skinny, angular features, blurry, distorted, extra limbs, deformed`

---

## Wan Video Prompt Rules
1. 80-120 words per 6-second clip
2. Structure: Subject → Action → Environment → Camera
3. ONE camera movement per clip
4. Clip 1 = setup, Clip 2 = payoff/punchline

---
