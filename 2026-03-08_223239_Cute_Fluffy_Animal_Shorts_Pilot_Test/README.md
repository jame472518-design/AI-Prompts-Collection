# Cute Fluffy Animal Shorts - Pilot Test Prompts

## Overview
Pilot test prompts for selecting a main character for a YouTube Shorts channel featuring cute, fluffy, chibi-style 3D animated animal characters.

## Animals Being Tested (7 candidates)
1. Fox (Red Fox) - 狐狸
2. Arctic Fox - 北極狐
3. Red Panda - 小熊貓
4. Hedgehog - 刺蝟
5. Rabbit (Lop-eared) - 垂耳兔
6. Baby Penguin - 企鵝寶寶
7. Raccoon - 浣熊

## Files
- `prompts_en.md` - English prompts (use directly in ComfyUI / Suno / Udio)
- `prompts_zh.md` - Chinese version with explanations for each prompt

## Per Animal
- 1 thumbnail image prompt
- 3 video scenarios (each = 2 clips x 6 sec = 12 sec)
- 3 music prompts (for Suno/Udio)

## Style
- 3D Pixar/Disney cartoon, chibi proportions
- Oversized head, tiny body, huge sparkly eyes
- Extremely fluffy plush-toy texture, kawaii aesthetic
- Pastel color palette

## Tools
- Video: Wan 2.1/2.2 (ComfyUI)
- Upscale: SeedVR2
- Music: Suno or Udio
